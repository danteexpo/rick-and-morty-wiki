# Rick & Morty Wiki

Project made with [The Rick and Morty API](https://rickandmortyapi.com/), Sass and Bootstrap.

## Link

[Rick & Morty Wiki](https://de-rick-and-morty-wiki.netlify.app/).

## Installation

```bash
npm install
```

Dependencies:

- bootstrap
- @popperjs/core --save
- sass
- react-paginate --save
- react-router-dom

## Start

```bash
npm start
```
